A lightweight Powershell database inspector with WinForms.

## Features

- [x] Connect to a database
- [x] Database support for SQL Server
- [x] WinForms GUI
- [x] Database aliases
- [x] Logging module
- [x] Query editor
- [x] Query history
- [ ] Performance analysis
- [ ] Index analysis
- [x] Backup manager
- [ ] Schema browser
- [ ] Schema visualization
- [x] Error handling with informative messages
- [x] Pester tests
- [ ] Documentation
