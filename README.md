A lightweight Powershell database inspector with WinForms.

## Features

- [x] Connect to a database
- [x] Database support for SQL Server
- [x] WinForms GUI
- [x] Database aliases
- [x] Logging module
- [ ] Query editor
- [ ] Query history
- [ ] Performance analysis
- [ ] Index analysis
- [ ] Backup manager
- [ ] Schema browser
- [ ] Schema visualization
- [ ] Error handling with informative messages
- [ ] Pester tests
- [ ] Documentation
