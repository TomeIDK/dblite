A lightweight Powershell database inspector with WinForms.

## Features

- [x] Connect to a database
- [x] Database support for SQL Server
- [x] WinForms GUI
- [x] Database aliases
- [x] Logging module
- [x] Query editor
- [x] Query history
- [x] Performance analysis
- [x] Index analysis
- [x] Backup manager
- [x] Schema browser
- [ ] Schema visualization
- [x] Error handling with informative messages
- [x] Pester tests
- [ ] Documentation
